# Change Log

All notable changes to the "debate-agent-mcp" extension will be documented in this file.

## [1.0.0] - 2024-12-07

### Added
- Initial release
- Multi-agent debate code review with P0/P1/P2 severity scoring
- Automatic MCP server configuration for VS Code
- Support for configurable agents (codex, claude, etc.)
- MCP tools: `list_agents`, `read_diff`, `run_agent`, `debate_review`, `debate_plan`
- Settings for default agents and auto-configuration
- Commands: Configure MCP Server, Show Status
